### 环境搭建

1: 下载安装thrift-0.9.3

2: 下载thrift 0.9.3源码

```
git clone -b 0.9.3 --single-branch https://github.com/apache/thrift.git
```

3: 复制thrift项目lib/java目录下的文件到本项目
